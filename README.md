# Tic-Tac-Toe Reinforcement Learning

Small project that trains a Q-learning agent to play Tic-Tac-Toe and evaluates performance against a random baseline.

## Results (10,000 games each)

### Before training
- **QLearningAgent vs RandomAgent**
  - QLearningAgent wins: **5777** (57.77%)
  - RandomAgent wins: **2972** (29.72%)
- **RandomAgent vs QLearningAgent**
  - RandomAgent wins: **5961** (59.61%)
  - QLearningAgent wins: **2834** (28.34%)
  - Ties: **1205** (12.05%)

### After training
- **QLearningAgent vs RandomAgent**
  - QLearningAgent wins: **9687** (96.87%)
  - RandomAgent wins: **90** (0.90%)
  - Ties: **223** (2.23%)
- **RandomAgent vs QLearningAgent**
  - RandomAgent wins: **5818** (58.18%)
  - QLearningAgent wins: **2926** (29.26%)
  - Ties: **1256** (12.56%)
